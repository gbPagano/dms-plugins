## Install

### DMS Plugins UI

Settings → **Plugins** → **Browse** → install **plugins**.

### Manual

```bash
# clone repo
git clone https://github.com/Dadangdut33/dms-plugins.git 
# copy the plugin that you need / want
cp -r dms-plugins/{plugin_name} ~/.config/DankMaterialShell/plugins/
# restart dms
dms restart  
```
